FlgithGear Tu-154B-2 Liveries

https://github.com/ShFsn/tu154b_liveries


NOTE. The aircraft itself is licensed under GPL v.3.
The liveries in this repository are mostly not licensed under GPL, unless
else is said. All rights holders are respected.


To install all liveries:

1. Clone this repository or download the archive.
2. Remove the existing Liveries folder in Models.
3. Unpack the archive if it was downloaded this way.
4. Move new Liveries folder into Models folder of the aircraft.


To install individual liveries, download the XML and PNG (or DDS, JPG,
etc.) files of an individual livery and move them into Models/Liveries
folder of the aircraft.


Paintkit is not available yet.
